# Algorithms for encoding and compressing information

Набор реализаций классических алгоритмов кодирования и сжатия информации на Python. Проект предназначен для обучения, экспериментов и сравнения свойств разных подходов (коэффициент сжатия, скорость, память).

## Содержание
- Краткое описание
- Реализованные алгоритмы
- Структура репозитория
- Установка

## Краткое описание
Этот репозиторий собирает наглядные и читаемые реализации алгоритмов сжатия и кодирования, которые удобно изучать, тестировать и расширять. Код ориентирован на простоту и понятность, а не на максимальную производительность.

## Реализованные алгоритмы
- Хаффман (Huffman coding)
- Арифметическое кодирование (Arithmetic coding)

## Структура репозитория
- arithmetic_lab/
- huffman_lab/
- README.md

## Установка
1. Клонируйте репозиторий:

   git clone https://github.com/Luk4red/Algorithms-for-encoding-and-compressing-information.git
   cd Algorithms-for-encoding-and-compressing-information

2. (Опционально) Создайте и активируйте виртуальное окружение:

   python -m venv .venv
   source .venv/bin/activate   # macOS/Linux
   .\.venv\Scripts\activate    # Windows
